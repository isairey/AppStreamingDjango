# 🎵 Django Music Streaming

### Plataforma moderna de streaming musical desarrollada con Django

Aplicación web de música construida con **Django**, **Python**, **HTML/CSS**, y **JavaScript**, enfocada en ofrecer una experiencia rápida, moderna y elegante para escuchar música desde cualquier dispositivo.

# ⚙️ Estructura del Proyecto

```bash
music-streaming/
│
├── media/                # Archivos multimedia
├── static/               # CSS, JS e imágenes
├── templates/            # Plantillas HTML
├── music/                # Aplicación principal
├── users/                # Gestión de usuarios
├── db.sqlite3
├── manage.py
└── requirements.txt
🚀 Instalación
1️⃣ Clonar el repositorio
git clone https://github.com/isairey/django-music-streaming.git
cd django-music-streaming
2️⃣ Crear entorno virtual
python -m venv venv
Activar entorno virtual
Windows
venv\Scripts\activate
Linux / macOS
source venv/bin/activate
3️⃣ Instalar dependencias
pip install -r requirements.txt
4️⃣ Ejecutar migraciones
python manage.py migrate
5️⃣ Crear superusuario
python manage.py createsuperuser
6️⃣ Ejecutar servidor
python manage.py runserver
🌐 Acceso

Abrir en el navegador:

http://127.0.0.1:8000

Panel administrativo:

http://127.0.0.1:8000/admin
